.. py:function:: return_ring_atoms(adj_list, idx, start=None, ring_size=10, counter=0, avoid_set=None, convert=True)

   Return ring(s) that atom idx belongs to
   The algorithm spawns non-backtracking walks along the graph.
   If the walk encounters the starting node, that consistutes a cycle.


.. py:function:: return_rings(adj_list, max_size=20, remove_fused=True)

   Finds the rings in a molecule based on its adjacency matrix. Most of the work in this function is done by
   `return_ring_atom()`, this function just cleans up the outputs and collates rings of different sizes.

   :param adj_list: A sublist is contained for each atom holding the indices of its bonded neighbors.
   :type adj_list: list of lists
   :param max_size: Determines the maximum size of rings to return.
   :type max_size: int, default=20
   :param remove_fused: Controls whether fused rings are returned (False) or not (True).
   :type remove_fused: bool, default=True

   :returns: **rings** -- List of lists holding the atom indices in each ring.
   :rtype: list
